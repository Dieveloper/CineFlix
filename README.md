**CineFlix**

CineFlix es una plataforma web de streaming desarrollada como Proyecto de Fin de Grado. La aplicación permite a los usuarios registrarse, iniciar sesión y acceder a un catálogo de películas y series de nicho desde una interfaz sencilla e intuitiva.

Además, incorpora un sistema multiperfil que permite que varios usuarios compartan una misma cuenta manteniendo perfiles independientes. También dispone de un panel de administración para la gestión de usuarios, películas, series, capítulos y recursos multimedia.

**Tecnologías utilizadas**

-Frontend
    Vue.js 3
    Vite
    Vue Router
    Pinia
    Axios
    CSS3
-Backend
    ASP.NET Core Web API (.NET 10)
    Entity Framework Core 10
    SQLite
    Swagger

**Funcionalidades principales**

  -Registro e inicio de sesión de usuarios.
  -Sistema multiperfil.
  -Catálogo de películas y series.
  -Clasificación por géneros.
  -Reproducción de contenido multimedia.
  -Gestión de capítulos para series.
  -Panel de administración.
  -Gestión de imágenes y vídeos.
  -Requisitos previos

Para ejecutar la aplicación es necesario disponer de:

.NET 10 SDK
Node.js
npm
Git

-Instalación
    Clonar el repositorio
    git clone https://github.com/Dieveloper/CineFlix.git
    
-Ejecutar el backend
    cd backend
    dotnet restore
    dotnet run
    
-Ejecutar el frontend
    cd frontend
    npm install
    npm run dev
